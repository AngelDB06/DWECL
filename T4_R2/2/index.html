La programación web permite crear sitios dinámicos e interactivos que pueden adaptarse a las necesidades del usuario. Mediante el uso de HTML, CSS y JavaScript se puede controlar el contenido y la presentación de la información.

JavaScript es un lenguaje de programación muy versátil, que se ejecuta directamente en el navegador del usuario. Con él podemos manipular el contenido de la página, responder a eventos y realizar cálculos sin necesidad de recargarla.

El control del tamaño de fuente y la alineación del texto son aspectos importantes en el diseño web, ya que influyen en la legibilidad y en la experiencia del usuario.

Gracias a las funciones de JavaScript, podemos crear controles que permitan al usuario personalizar la apariencia del texto de manera sencilla e inmediata.
